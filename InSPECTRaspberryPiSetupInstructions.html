Prerequisites

We assume you have a Raspberry Pi running a recent version Raspbian (2016-11-25 or newer) with proper settings for language, keyboard, and timezone. We assume that your Pi is connected to the internet.

Linux Basics

You may find the commands useful:

  • ls: list files
  • ls -la: list all files and provide detailed info about them
  • cd dir: change directories
  • pwd: show present working directory
  • cat filename: display the contents of a file
  • nano filename: edit a text file

Installing the software

  1. Click the >_ icon at the top of the screen to get a terminal window.
  2. Make sure your system software is up to date:
  1. Run: sudo apt purge wolfram-engine
  2. Run: sudo apt-get autoremove
  3. Run: sudo apt-get clean
  4. Run: sudo apt update
  5. Run: sudo apt upgrade (enter y if prompted; this could take an hour or more depending on the speed of your network connection and how recent your version of Raspbian is; if you are in a hurry you can skip this step)
  1. Install dependencies:
  1. Run: sudo apt install libjpeg-dev
  2. Run: sudo apt install python-dev
  3. Run: sudo pip install gevent (This may take a few minutes and will display some warnings; no need to worry about them.)
  4. Run: sudo pip install ws4py
  5. Run: sudo pip install hjson
  1. Install rhizo:
  1. Run: git clone git://github.com/rhizolab/rhizo
  2. Run: nano .bashrc
  3. Add this line at the end of the file: export PYTHONPATH=/home/pi/rhizo
  4. Press ctrl-O, ctrl-X to save and exit
  5. Run: source .bashrc
  1. Install flow: git clone git://github.com/manylabs/flow
  2. Run flow:
  1. Run: cd flow
  2. Run: python flow.py (If you run into an error about being unable to import rhizo.main, close and re-open your terminal window then start again with step 6.)
  3. Note the PIN displayed by the program.
  1. Create a controller and associate it with your Raspberry Pi:
  1. Go to the dataflow.manylabs.org website.
  2. Log in.
  3. Click a folder name from the top menu bar (e.g. Testing).
  4. Select New (drop down menu near top of page) and then Controller to create a new controller.
  5. Select Enter PIN from the drop down menu for the controller you just created. (If you don’t see a drop down menu, select View as Tiles.)
  6. Enter the PIN displayed in step 5.
  1. Try the flow interface
  1. Go to https://dataflow.manylabs.org/ext/flow
  2. Click the name of your controller.
  3. Click New Diagram.
  4. Click Add Sim Sensor (or plug in a real sensor).
  5. The web interface should show a block with a number.
  1. Set up the flow program to run on startup:
  1. Run:  chmod +x /home/pi/flow/flow.sh
  2. Run:  nano /home/pi/.config/lxsession/LXDE-pi/autostart
  3. Add this line: @lxterminal -e "/home/pi/flow/flow.sh"
  4. Restart the RPi to test the startup code.

Running the data flow program

To run the data flow program after turning on your Raspberry Pi, follow these steps:

  1. Click the >_ icon at the top of the screen to get a terminal window.
  2. Run: cd flow
  3. Run: python flow.py

Upgrading the data flow program

You can upgrade your copies of the flow program and rhizo library to the latest versions using these steps:

  1. Click the >_ icon at the top of the screen to get a terminal window.
  2. Run: cd flow
  3. Run: git pull
  4. Run: cd (changes back to home directory)
  5. Run: cd rhizo
  6. Run: git pull
  7. Run: cd

Embedding in CODAP

  1. log out of the dataflow server (to replicate student experience; students won’t be logged in)
  2. go here: http://codap.concord.org/releases/latest/static/dg/en/cert/index.html
  3. select Create New Document
  4. select Import from the top-left menu then URL
  5. enter this URL: https://dataflow.manylabs.org/ext/flow?codap=1
  6. two ways to transfer data to CODAP:
  1. click the Explore Data in CODAP item in the menu for a plotting block
  2. click the View Recorded Data item in the menu for a sensor block (after recording some data) and then click Explore Data in CODAP (you can optionally select an interval)
  1. in the CODAP interface, click the Tables icon and select one of the data tables
